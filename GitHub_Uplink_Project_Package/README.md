# Satellite Communications Project - Uplink

## Description
This repository contains the Uplink simulation/project for the Satellite Communications course (Academic Year 2025-2026).

## Contents
- `Uplink.py` → Main GUI application (unchanged original source code)
- `Project_Report_English.pdf` → English version of the project report

## Requirements
Install dependencies with:

```bash
pip install -r requirements.txt
```

## Run the Project

```bash
python Uplink.py
```
